# DIKWP Agent LearnLab V1

Standalone, offline-first prototype for learning AI agent development and deployment.

## Workflow

Agent concept map → M0–M5 learning path → Use Case Canvas → DIKWP Agent Loop → Tool/MCP Registry → RAG and Memory → Orchestration Patterns → AgentTrace → EvalOps → Deployment Pipeline → Security Threat Model → Incident Simulation → Agent Learning Passport.

## Quick start

Open `index.html` in a browser. No server, database, model API, account, or network is required.

Optional CLI:

```bash
python run_demo.py examples/sample_agent_project.json --out outputs
python toy_agent_runtime.py examples/sample_trace.json
python static_boundary_audit.py
```

## Boundaries

The prototype does not call external models, execute production tools, write to external systems, or provide compliance certification. It is a learning and architecture sandbox.
